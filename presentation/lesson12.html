Object Oriented Programming (OOP)

in JavaScript(ES2020)

  • Object-Oriented Programming - a concept used by programmers to develop fully independent units of code. Each independent unit is known as an object.
  • Object - something that has properties and methods.

Class

  • Class - "recipe" where objects derive their abilities.
  • You can make your own classes with their own properties and methods.
  • Class is not used directly but needs instance = object which we can use and manipulate.
  • in OOP we speak also about abstract class - can't be instiantiated, serves only for inheritance purposes. (‚ùåJS ‚úÖTS)

Interface, visibility

  • Interface - "recipe" which properties should object have. Can serve as implementation requirement for class. (‚ùåJS ‚úÖTS)
  • Properties and Methods can have prefixes

JS implementation


class SomeClass {
  #propPrivate;
  static SOME_CONST = 5;
  constructor() {
    this.propPublic = 'pub';
    this.#propPrivate = 'priv';
  }
  static methodStatic() { console.log(this.SOME_CONST + 1); }
  methodPublic() { console.log('public'); }
  #methodPrivate() { console.log('private'); }
}

JS implementation


SomeClass.SOME_CONST
// 5
SomeClass.methodStatic()
// 6
let obj = new SomeClass();
obj.propPublic
// "pub"
obj.methodPublic()
// "public"
obj.#propPrivate
// SyntaxError: Private field '#propPrivate' must be declared
// in an enclosing class

Inheritance

As any person inherits features from their parents, classes can act the same way. You can create a class that inherits its capabilities from another class.

Inheritance


class Mammal {
  constructor() {
    this.legs = 4;
  }
  doSound() { console.log('Not yet implemented'); }
}

class Dog extends Mammal {
  constructor() { super(); }
  doSound() { console.log('wuff wuff'); }
}

class Ape extends Mammal {
  constructor() { super(); this.legs = 2; }
  doSound() { console.log('huhuhu hahaha'); }
}

JS implementation


let husky = new Dog();
husky.legs
// 4
husky.doSound()
// wuff wuff
let judy = new Ape();
judy.legs
// 2
judy.doSound()
// "huhuhu hahaha"
